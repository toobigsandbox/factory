Geometry // Volume Foundations

Building Volume with Unit Cubes

Measuring Space via One-to-One Correspondence

Material Analysis

Box of 2-centimeter unit cubes, various geometric solids (cubes/prisms from the pink tower or elementary set), tickets and labels.

Presentation I: Comparison

Build two different solids using unit cubes. Make one 10 cubes and the other 12 cubes.
"Which of these takes up more space? Remember, we use squares for area, but for solids, we use cubes. Let's count by twos."

Presentation II: Recreation

Invite the child to pick a cube from the pink tower and recreate it using unit cubes.
"First, we build the floor—the base layer. Then we add more layers until it's the same height. Each layer has the same number of cubes."
"It is vital the child builds visibly. They must be able to count every cube. Do not let them build 'hidden' spaces that they can't verify sensorially."
Elim Lee // Semester 4 Sprint // AMI Elementary
