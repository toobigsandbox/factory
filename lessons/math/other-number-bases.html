Mathematics // Advanced Number Study

Other Number Bases (3 & 5)

Guide: Elim Lee // AMI Elementary Submission

Materials

Bead bars, Wooden base conversion cards, Graph paper.

Activity I: Counting in Base 3

Place one unit bead bar.
"In Base 3, we only have three symbols: 0, 1, and 2. Once we reach 3, we move to the next wire."
Fig 1.1: Stick figure counting bead bars in Base 3

Activity II: Converting Decimal 17

Convert 17 to Base 5 using the repeated division method.
"We ask: How many fives are in 17? Three, with a remainder of two."
Fig 2.1: Converting 17 to Base 5
Repeat for Base 3.
"For Base 3: 17 ÷ 3 = 5 (R2), 5 ÷ 3 = 1 (R2), 1 ÷ 3 = 0 (R1). The result is 122."
